# BankingApplication
## Project for COP 4331 - using Flask, Gunicorn, MySQL, Docker.

Build instructions: 
```
  git clone https://github.com/hsauers5/BankingApplication
  sudo apt update
  sudo apt install docker
  sudo apt install docker-compose
  docker-compose build
  docker-compose up
```
